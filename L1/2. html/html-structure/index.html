HTML Document Structure

This diagram shows how HTML elements are nested to form a complete semantic layout.

<html>
<head>

Holds non-visible data like charset, title, styles, SEO

<meta>

Defines page settings and encoding

<title>

Shown on browser tab or bookmark

<body>

Everything visible to the user

<header>

Introductory section of the page

<nav>

Main navigation links

<main>
<section>
<article>

A standalone piece of content

<aside>

Extra info, ads, links, or quotes

<footer>

Contact info, links, copyright
