The CSS Box Model

Every HTML element is a box! This diagram shows the four parts of the box model: Content, Padding, Border, and Margin.

MARGIN

Transparent space outside the border, separating this element from others.

BORDER

The line around the padding and content. It has width, style, and color.

PADDING

Space between the content and the border. It takes the background color of the element.

CONTENT

This is where your actual text, images, or other media reside.
